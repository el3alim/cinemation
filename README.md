# Cinemation

A movie overview showcase website

# Features

* ReactJS
* TailwindCSS
* Mobile Responsiveness
* ENV
* Movies information; Current, Upcoming, Popular, Rating, IMDB
* TMDB / Movie DB API
* Search Movies
* Light / Dark Mode
