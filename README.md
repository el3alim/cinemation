# Cinemation

A movie overview showcase project

# Features

* Movies information; Current, Upcoming, Popular, Rating, IMDB
* TMDB / Movie DB API
* Search Movies
* Routes
* ENV
* TailwindCSS
* Light / Dark Mode
* Mobile Responsiveness
